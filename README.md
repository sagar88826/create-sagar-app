## Quick Start

```bash
# Using npm
npx create-sagar-app my-app

# Using yarn
yarn create sagar-app my-app

# Using pnpm
pnpm create sagar-app my-app
```

## Requirements

- Node.js 18 or later

## Development

After creating your project:

```bash
cd my-app
pnpm install
pnpm dev
```

This will start both the client and server in development mode.
